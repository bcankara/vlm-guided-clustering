## Understanding the Experiments

### What's in Each Experiment Folder?

Each `vlm_*` folder contains:

| File/Folder | Description |
|-------------|-------------|
| `step_01/`, `step_02/`, ... | Visualizations and JSON data for each VLM decision step |
| `final/` | Final cluster visualizations and merge round summaries |
| `llm_guided/` | Intermediate results (final_results.json, flow_tree.json) |
| `experiment_log.json` | Complete experiment log |
| `experiment_report.md` | Auto-generated markdown report |
| `cluster_hierarchy.txt` | Text-based cluster tree |

### Key Metrics

- **ARI (Adjusted Rand Index):** Measures clustering accuracy (1.0 = perfect)
- **NMI (Normalized Mutual Information):** Measures information overlap (1.0 = perfect)
- **K:** Number of final clusters (ground truth = 4)

## Phase Descriptions

### Phase 0: Baby Steps (December 26, 2025)

The project's **earliest experiments**. Key learnings:

- `first_vlm_experiment/`: First attempt at VLM-guided clustering (K=6, slight over-split)
- `over_split_example/`: Severe over-splitting case with K=30 (!)
- This phase revealed that raw visualizations confused the VLM

**Typical Results:** ARI ~0.30-0.50, K values 6-30

### Phase 1: Baseline (December 26-28, 2025)

Initial structured experiments with proper tracking:

- Basic prompts without specific guidance
- No visualization normalization
- VLM treated amplitude differences as behavioral differences

**Typical Results:** ARI ~0.48, K values 10-23

### Phase 2: Zone Verification (December 29-31, 2025)

Major improvements with zone-based verification:

- Added vertical reference lines at x=25, 50, 75, 100, 125
- Introduced behavior classification system
- K-Shape algorithm achieved ARI=0.99

**Typical Results:** ARI ~0.61-0.99, K values 4-8

### Failed Experiments: Over-Merge Cases

Cases where VLM incorrectly merged clusters:

- Final K=3 instead of ground truth K=4
- VLM gave 100% confidence on wrong decisions
- All failures are Hierarchical algorithm

**Lesson:** High confidence ≠ correctness

## How to Reproduce

1. **Run the main experiment:**
   ```bash
   python main.py
   ```

2. **Select experiment type (menu):**
   - Options 1-3: Baseline algorithms
   - Options 4-6: VLM-guided algorithms
   - Option 7: Fixed K=4 comparison
   - Option 8: Reproducibility test (6x runs)

3. **Check results in:**
   - `Results/` folder for current runs
   - `Gemini_*/` folders for model-specific results
